# 💰 حاسبة المدخرات السعودية

أول تطبيق ويب سويته بجوال أندرويد. يحسب مدخراتك الشهرية والسنوية ويحفزك توصل لأهدافك المالية.

### ✨ المميزات
- حساب المدخرات الشهرية والسنوية تلقائيًا
- رسوم بيانية توضح نسبة المصاريف vs المدخرات  
- رسائل تحفيزية: "ممتاز يا عزالدين! تقدر تستثمر بقوة"
- توقع المبلغ بعد 1، 3، 5، 10 سنوات
- حاسبة الهدف: كم شهر باقي عشان توصل لهدفك؟

### 🛠️ التقنيات المستخدمة
- **React + TypeScript** - واجهة تفاعلية سريعة
- **Framer Motion** - أنيميشن للأرقام
- **Recharts** - رسوم بيانية احترافية
- **Tailwind CSS** - تصميم عصري متجاوب مع الجوال
